This folder contains the following files:
1. totalType.txt
	Each line represents an entity of YAGO. Each line starts with the id, following by a list of type ids.
	
2. yagoadj.txt
	Each line represents an edge of YAGO, with format "sid	eid	edgeType".
	
3. yagoTaxID.txt
	Each line matches an entity id to its name.
	
4. yagoType.txt
	Each line matches an edge type to its name.
